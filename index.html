About Bioconductor

version 1.0.0 (2021-07-27)

Bioconductor provides tools for the analysis and comprehension of high-throughput genomic data (Huber et al. 2015).

Bioconductor uses the statistical programming language (R Core Team 2021), and is open source and open development. It has two releases each year, and an active user community.

Bioconductor is also available as Docker images.
